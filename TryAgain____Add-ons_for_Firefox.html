    About this Add-on

    TryAgain keeps trying to load a webpage when the server cannot be reached. Sometimes you know that you've entered the address correctly, but the page won't load the first time around. Maybe your wireless connection dropped or the server hosting the website is temporarily down; whatever the reason, TryAgain will try and try again.

    Developer’s Comments

    This extension is based on a Greasemonkey script made by Paul Lammertsma. That original script is now obsolete as he has joined the TryAgain developer team!
